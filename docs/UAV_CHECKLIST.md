# User Acceptance Verification (UAV)
## MS Accounting System

> **متى تُشغَّل؟** بعد أي Release مهم أو Merge كبير.
> **من يُشغّلها؟** المستخدم الفعلي — ليس الكود، ليس الـ API.
> **الهدف:** التأكد أن النظام يعمل من منظور المستخدم النهائي.
>
> ⏱ الوقت المتوقع: **20-30 دقيقة**
> 🌐 الرابط: https://mosaeed1099-ship-it.github.io/ms-accounting

---

## طريقة الاستخدام

لكل سيناريو:
- **✅** — يعمل بشكل طبيعي
- **❌** — فيه مشكلة (اكتب ملاحظة)
- **⏭** — تخطيت (مع السبب)

في نهاية الجلسة أكمل قسم **UAV Report** في الأسفل.
