# Regression Testing Checklist — MS Accounting
> نفّذ هذه الاختبارات قبل كل Deployment. إذا فشل أي اختبار → أوقف الـ Deploy فوراً.

---

## كيف تستخدم هذه القائمة

1. افتح الموقع على **mobile** و**desktop** معاً
2. سجّل الدخول كـ Admin
3. نفّذ كل اختبار بالترتيب
4. ✅ = نجح | ❌ = فشل (اكتب الخطأ)
5. إذا كان هناك ❌ واحد → **لا تنشر**
